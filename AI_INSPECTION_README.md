## 🎯 使用技巧

### ✅ 最佳拍摄方式
```
✓ 光线充足均匀
✓ 正面拍摄产品
✓ 纯色背景
✓ 图片清晰（≥1280x720）
✗ 避免强光反射
✗ 避免倾斜角度
```

### ✅ 最佳工作流程
```
1. 启动连续检测模式
2. 按固定节奏放置产品
3. 自动拍摄并检测
4. 实时查看统计数据
5. 班次结束生成日报表
```

### ✅ 瑕疵率管理
```
< 2%   🟢 优秀 - 继续保持
2-3%   🟢 良好 - 维持现状
3-5%   🟡 注意 - 需要改进
> 5%   🔴 警告 - 立即检查
```

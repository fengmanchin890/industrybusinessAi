## 🚀 使用方法

### 開發環境
```bash
npm run dev
# 應用會自動在瀏覽器中打開 http://localhost:5173
```

### 生產構建
```bash
npm run build
# 優化後的構建會包含：
# - 代碼分割
# - Tree shaking
# - 壓縮優化
# - Console 移除
```

### 預覽生產構建
```bash
npm run preview
# 在 http://localhost:4173 預覽
```

## 🔧 技術棧

- **框架**: React 18.3+ with TypeScript
- **構建工具**: Vite 5.4+
- **樣式**: Tailwind CSS 3.4+
- **圖標**: Lucide React
- **後端**: Supabase
- **動畫**: CSS 自定義動畫 + Tailwind 過渡
